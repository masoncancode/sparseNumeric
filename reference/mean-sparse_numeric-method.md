# Mean of a sparse_numeric vector

Computes the mean of the underlying full vector (including implicit
zeros) without coercing to a dense representation.

## Usage

``` r
# S4 method for class 'sparse_numeric'
mean(x, ...)
```

## Arguments

- x:

  A
  [sparse_numeric](https://masoncancode.github.io/sparseNumeric/reference/sparse_numeric-class.md)
  object.

- ...:

  Ignored.

## Value

Numeric scalar mean.
